# sandlot-bentobox-tools

BentoBox addon: admin island-state recovery and blueprint regen for BSkyBlock.

This repository contains the v1 addon referenced by admindoc's feature spec `specs/2026-05-25-sandlot-bentobox-tools/feature.md`.

Build:

```bash
mvn clean package
```

Drop `target/sandlot-bentobox-tools-0.1.0-SNAPSHOT.jar` into your BentoBox server's `addons/` directory and restart the server.
